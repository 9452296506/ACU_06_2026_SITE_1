# Changelog

All notable changes to the **Beauty Age** landing page are documented here.
This project follows [Semantic Versioning](https://semver.org/) (`MAJOR.MINOR.PATCH`).

---

## [1.0.2] — 2026-06-17

### Improved
- Minor copy and EU cosmetic-compliance wording improvements across sections and FAQ
- Cart UI improvements (quantity controls, subtotal formatting, checkout demo modal)

---

## [1.0.1] — 2026-06-16

### Improved
- Refined mobile layout (single-column sections, full-width CTAs, near full-screen cart drawer)
- Improved logical use of imagery from `images/` (hero, product card, ritual, expert, testimonials, UGC, before/after)
- Accessibility refinements (ARIA labels on icon buttons, keyboard-accessible FAQ, focus states, ESC/overlay close)

---

## [1.0.0] — 2026-06-15

### Added
- Initial Beauty Age landing page (Spanish, es-ES)
- Responsive layout (breakpoints at 1200 / 992 / 768 / 480 px)
- Sticky header with announcement bar, search, account and cart icons; slide-in mobile menu
- Hero with product + model imagery, benefit bullets and price block (39 € · 50 ml)
- Sections: problem / skin needs, product card, ingredients, how-to-use ritual, benefits, expert / quality, testimonials, illustrative changes, UGC gallery
- Shopify-like cart drawer with `localStorage` persistence
- Search modal with live filtering (product, sections, FAQ)
- Account modal (login / register demo tabs)
- FAQ accordion
- Contact form with client-side validation and privacy consent
- Newsletter form with privacy consent
- Cookie banner (Accept / Configure) with `localStorage` consent
- Policy modals: Privacy, Terms, Delivery, Returns, Cookies
- EU cosmetic claim cautious wording (Regulation (EC) No 1223/2009 and Regulation (EU) No 655/2013)
- SEO meta tags, Open Graph tags, JSON-LD `Product` structured data, canonical and favicon placeholders
