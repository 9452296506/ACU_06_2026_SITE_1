## Deployment compatibility

The project is a static website and is compatible with any standard web-hosting environment:

- Shared / VPS hosting with **Apache** or **Nginx**
- Static hosts: **Netlify, Vercel, GitHub Pages, Cloudflare Pages**
- CDN-based distribution

No backend, no database and no build pipeline are required. Deployment is performed by uploading the contents of the project root directory to any web server. The site also runs locally by opening `index.html`.

## Source code / repository policy

The production source code can be kept in a **private repository** and shared directly with the client, while a public repository holds only the delivery documentation and visual proof of delivery. Access management is performed on the client's request.
